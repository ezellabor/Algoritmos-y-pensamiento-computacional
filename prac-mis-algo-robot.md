# Propuesta Formal – Misión Algoritmia

## Resumen Ejecutivo
Misión Algoritmia es un juego didáctico para enseñar pseudocódigo y Java a alumnos de ASIR1 y SMR1 (robótica), mediante misiones con un robot simulado.

## Competencias trabajadas
- Secuencias, condicionales y bucles
- Variables y contadores
- Traducción pseudocódigo → Java
- Depuración lógica
- Algoritmos para control de robots y lectura de sensores

## Plan de implantación
- Pilotaje: 4 sesiones de 60 min (introducción, condicionales, bucles, mini-proyecto)
- Material: prototipo web, fichas de misiones, rúbrica de evaluación

## Recursos necesarios
- Ordenadores con navegador, opción proyector, opcional kits robóticos para integración

## Evaluación
- Rúbrica (ver archivo `rubrica.md`) con criterios: Funcionalidad, Uso de Estructuras, Claridad, Optimización


# Rúbrica de Evaluación – Misión Algoritmia

Puntuación máxima: 12 puntos (4 criterios × 3 puntos)

| Criterio | 1 - Insuficiente | 2 - Aceptable | 3 - Excelente |
|---|---|---|---|
| Funcionalidad | No cumple objetivo ni funciona | Cumple objetivo con errores menores | Cumple objetivo de forma robusta |
| Uso de estructuras | No usa bucles/condicionales cuando toca | Usa estructuras correctamente | Usa y optimiza estructuras |
| Claridad pseudocódigo | Confuso o erróneo | Legible | Muy claro, con comentarios |
| Optimización | Redundante | Correcto | Eficiente y elegante |

**Conversión orientativa:** 10–12 = Sobresaliente, 7–9 = Notable, 5–6 = Aprobado, <5 = Suspenso
